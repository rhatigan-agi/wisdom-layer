# Contributing

Thanks for your interest in Wisdom Layer.

## What we accept

- **Bug reports** via [Issues](https://github.com/rhatigan-agi/wisdom-layer/issues)
- **Feature ideas** via [Discussions](https://github.com/rhatigan-agi/wisdom-layer/discussions)
- **Documentation issues** via Issues (typos, unclear sections, broken links)
- **Questions and showcases** via Discussions

## What we don't accept

The Wisdom Layer SDK source is developed in a private repository and shipped as a compiled wheel. We don't currently accept external code contributions — pull requests will be politely closed.

This may change in the future. If you have a substantial idea that would require code, please open a Discussion first so we can talk it through.

## Reporting security issues

See [SECURITY.md](SECURITY.md). Please don't report security issues via public Issues.

## Code of conduct

Discussions and Issues are governed by our [Code of Conduct](CODE_OF_CONDUCT.md). Be kind, be specific, and assume good faith.
